## Format

- PNG, JPG veya SVG formatında olabilir
- Önerilen boyut: 100x100px veya 200x200px
- Şeffaf arka plan tercih edilir

## Kullanım

Logo dosyalarını bu klasöre ekledikten sonra, `index.html` dosyasındaki `clubLogos` dizisindeki dosya yolları otomatik olarak çalışacaktır.
